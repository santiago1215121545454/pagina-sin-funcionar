Curso cmd (parte 1)

hola hackers de informatica online. Hoy les traigo un tutorial
del cmd (parte 1).

primero veremos el comando color, este comando sirve para perzonalizar la consola de comandos en cuanto a color
se refiere, en mi caso puse letras verdes. Para investigar ingresen el comando color/? este comando lo pueden
usar para investigar cualquier comando /?.


ahora veremos el comando cd, este comando sirve para abrir carpetas.


el comando md, nos sirve para crear una carpeta, importante siempre al momento de crear una carpeta desde el cmd pongan el nombre
entre comillas.


mientras que el comando rd nos sirve para eliminarlas.


otro comando que nos sirve para navegar entre carpetas, es cd.. que sirve para regresar una carpeta atras.


y el ultimo comando que veremos hoy, es dir, que nos sirve para ver el contenido de una carpeta.

antes de terminar el tutorial, les quiero compairtir un comando muy sencillo pero a la vez muy util, y es cls
este comando sirve para limiar todos los comandos que hemos escrito anteriormente.
